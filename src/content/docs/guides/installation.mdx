---
title: Installation
description: How to install Nokkvi on your system.
---

Nokkvi is currently designed and tested for **Linux only** (specifically Arch Linux).

## Dependencies

You will need `pipewire`, `fontconfig`, and `pkg-config` installed on your system.

```bash
# Arch Linux
sudo pacman -S pipewire fontconfig pkg-config
```

## Building from source

1. Clone the repository:
   ```bash
   git clone https://github.com/f-o-o-g-s/nokkvi.git
   cd nokkvi
   ```

2. Build the release binary:
   ```bash
   cargo build --release
   ```

3. Run the installer:
   ```bash
   ./install.sh
   ```

This will copy the binary to `~/.local/bin/nokkvi` and set up the desktop entry and icons.
