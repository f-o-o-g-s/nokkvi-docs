The Albums view is Nokkvi's main library browser. Each row shows the cover, title, and album artist; the right side shows a large artwork panel for the focused album with a metadata overlay.

For navigation, search, multi-select, ratings, and shared right-click actions, see [Library Basics](/guides/library-basics/).

## Sort modes

- **Recently Added** *(default)*, **Recently Played**, **Most Played**, **Favorited**, **Random**
- **Name**, **Album Artist**, **Artist**, **Release Year**
- **Song Count**, **Duration**, **Rating**, **Genre**

The dynamic column on the right adapts to the current sort — release year, duration, last play date, or genre.

## Columns

Toggle via the gear icon: **Song Count**, **Stars** (rating), **Plays**, and **Love** (heart). Stars and Plays auto-show when you sort by them.

## Expanding to tracks

`Shift + Enter` (or clicking the "*N* songs" text) expands the focused album so its tracks appear indented below it. Only one album can be expanded at a time; changing sort, search, or refreshing collapses it.

Both album rows and expanded track rows offer the standard **Add to Queue / Add to Playlist / Get Info / Show in File Manager** context-menu actions.

## Click-through expansion

- **Album artist** on every row → Artists view with that artist's albums expanded inline at the top.
- **Genre** in the dynamic column when sorted by Genre → Genres view with that genre expanded inline.

Collapse with `Shift + Enter`, or refresh / change sort to clear the expansion state.

## Artwork refresh

The right-side artwork panel has a refresh button that re-pulls cover art from the server, evicting the local cache. Useful when you've updated artwork in Navidrome and want to see it immediately.
