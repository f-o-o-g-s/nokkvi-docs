Nokkvi is designed to be fully navigable via keyboard. Every action below can be rebound ‚Äî see [Customizing Shortcuts](#customizing-shortcuts) at the bottom of this page.

## Views

Switch between the different screens of the application.

| Action | Shortcut | Description |
| :--- | :--- | :--- |
| Queue | `1` | Active playback queue |
| Albums | `2` | Browse library by album |
| Artists | `3` | Browse by artist |
| Songs | `4` | Flat, searchable track list |
| Genres | `5` | Browse by genre |
| Playlists | `6` | Navidrome playlists |
| Radios | `7` | Radio stations |
| Harbour | `8` | Home view ‚Äî discovery shelves and whole-library search |
| Settings | `` ` `` (Backtick) | Open the settings panel |

## Playback

| Action | Shortcut | Description |
| :--- | :--- | :--- |
| Play / Pause | `Space` | Play or pause the current track |
| Toggle Random | `X` | Toggle random/shuffle mode |
| Toggle Repeat | `Z` | Cycle repeat mode: off ‚Üí one ‚Üí queue |
| Toggle Consume | `C` | Consume mode removes each track from the queue after it plays |
| Toggle SFX | `S` | Enable or disable UI sound effects |
| Cycle Visualizer | `V` | Cycle through visualizer modes: off ‚Üí bars ‚Üí lines ‚Üí scope |
| Toggle Equalizer | `Q` | Open or close the 10-band graphic equalizer |
| Open Trawl | `t` | Open the [Trawl](/guides/trawl/) mix-builder modal (works in any view except Settings) |
| Toggle Crossfade | `F` | Enable or disable gapless crossfading between tracks (enabling it turns off bit-perfect) |
| Toggle Lyrics | `L` | Show or hide [synced lyrics](/guides/lyrics/) over the Queue cover art |
| Toggle Bit-Perfect | `B` | Cycle bit-perfect output: off ‚Üí strict ‚Üí relaxed (turns off crossfade) |

## Navigation

The slot list is the scrollable item list in the center of each view.

| Action | Shortcut | Description |
| :--- | :--- | :--- |
| Slot List Up | `Backspace` | Move the centered slot up in the list |
| Slot List Down | `Tab` | Move the centered slot down in the list |
| Activate / Enter | `Enter` | Play or open the focused item |
| Shuffle Play | `Ctrl + Enter` | Shuffle-play the focused item or selection (forced shuffle, ignores the Enter setting) ‚Äî or, in the [Trawl](/guides/trawl/#keyboard) modal, play the mix (replaces the queue) |
| Expand / Collapse | `Shift + Enter` | Expand / collapse the focused item inline (albums, artists, playlists, genres) |
| Search | `/` | Focus the search bar in the current view |
| Center on Playing | `Shift + C` | Scroll the list to the currently playing track |
| Library Browser | `Ctrl + E` | Toggle the library browser panel alongside the Queue |

## Item Actions

| Action | Shortcut | Description |
| :--- | :--- | :--- |
| Add to Queue | `Shift + A` | Append the focused item (album, song, artist, genre, or playlist) to the playback queue ‚Äî or, in the [Trawl](/guides/trawl/#keyboard) modal, add the whole mix to the queue |
| Remove from Queue | `Ctrl + D` | Remove the focused item from the queue |
| Clear Queue | `Shift + D` | Remove all tracks from the queue |
| Love / Unlove | `Shift + L` | Toggle the Navidrome star ("love") flag on the focused track |
| Increase Rating | `=` | Increase the star rating by one |
| Decrease Rating | `-` | Decrease the star rating by one |
| Get Info | `Shift + I` | Open the info panel for the focused item |
| Find Similar | `Shift + S` | Find songs similar to the currently playing track |
| Top Songs | `Shift + T` | Show top songs for the playing track's artist |
| Move Track Up | `Shift + Up` | Move the focused queue track up by one position |
| Move Track Down | `Shift + Down` | Move the focused queue track down by one position |
| Save Queue as Playlist | `Ctrl + S` | Open the save-as-playlist dialog for the current queue |
| New Smart Playlist | `Shift + N` | Open the [smart-playlist](/guides/smart-playlists/) rules editor on a new blank playlist |
| Edit Centered Playlist | `E` | Edit the centered playlist (Playlists view) ‚Äî rules editor for smart, track editor for manual |
| Trawl: Save Mix as Playlist | `Shift + P` | Save the current [Trawl](/guides/trawl/) mix as an ordinary (static) playlist ‚Äî mix builder only |

## Sort & View

| Action | Shortcut | Description |
| :--- | :--- | :--- |
| Previous Sort Mode | `Left` | Cycle the sort mode backward ‚Äî or step the focused [Trawl](/guides/trawl/#editing-the-controls-from-the-keyboard) control to its previous value |
| Next Sort Mode | `Right` | Cycle the sort mode forward ‚Äî or step the focused Trawl control to its next value |
| Toggle Sort Order | `Page Up` | Toggle ascending / descending sort |
| Refresh View | `R` | Reload the current view's data from the server |
| Roulette | `Ctrl + R` | Spin the wheel and play a random item from the current view |

## Global

| Action | Shortcut | Notes |
| :--- | :--- | :--- |
| Escape / Back | `Escape` | Close overlays, clear search, or go back ‚Äî **not rebindable** |
| Reset to Default | `Delete` | Reset the focused setting to its default value (Settings view only) ‚Äî **not rebindable** |

## Settings Navigation

`Up` / `Down` edit values only in the Settings view. `Shift + Tab` / `Shift + Backspace` move between settings categories here, and do double duty inside the [Trawl](/guides/trawl/#editing-the-controls-from-the-keyboard) modal, where they step the controls-row focus ring.

| Action | Shortcut | Description |
| :--- | :--- | :--- |
| Edit Value Up | `Up` | Toggle a setting on, or increment a value |
| Edit Value Down | `Down` | Toggle a setting off, or decrement a value |
| Next Settings Category | `Shift + Tab` | Move to the next category in the settings sidebar ‚Äî or step the [Trawl](/guides/trawl/#editing-the-controls-from-the-keyboard) controls-row focus ring forward |
| Previous Settings Category | `Shift + Backspace` | Move to the previous category in the settings sidebar ‚Äî or step the Trawl controls-row focus ring backward |

## Smart playlist rules editor

The [smart-playlist rules editor](/guides/smart-playlists/#the-rules-editor) is keyboard-first. Most of its keys are fixed context behaviors rather than rebindable actions ‚Äî the exceptions are **Save**, which follows your *Save Queue as Playlist* binding (`Ctrl + S` by default), and the two entry-point actions above (`Shift + N`, `E`).

Building the rules (form pane):

| Action | Shortcut |
| :--- | :--- |
| Move between rows | `Up` / `Down` |
| Step or cycle the focused cell | `Left` / `Right` |
| Next cell in the row | `Tab` |
| Switch between the form and the preview | `Shift + Tab` |
| Act on the focused cell ‚Äî open a picker, toggle, add or remove a rule | `Enter` |
| Reorder a sort key | `Shift + Up` / `Shift + Down` |
| Remove the focused rule or sort key | `Delete`, `Backspace`, or `x` |
| Preview (re-evaluate on the server) | `Ctrl + Enter` |
| Save | `Ctrl + S` |
| Discard (confirms if there are unsaved edits) | `Escape` |

While editing a text or number cell, `Enter` or `Tab` commits it and `Escape` reverts just that cell. In the preview pane, `Up` / `Down` move and `Enter` plays the evaluated list from the focused row.

Pickers and the calendar:

| Action | Shortcut |
| :--- | :--- |
| Filter a list picker (field, operator, genre, playlist‚Ä¶) | *type to filter*, then `Up` / `Down`, `Enter`, `Escape` |
| Calendar ‚Äî move by a day | `Left` / `Right` |
| Calendar ‚Äî move by a week | `Up` / `Down` |
| Calendar ‚Äî previous / next month | `Page Up` / `Page Down` |
| Calendar ‚Äî pick the focused day | `Enter` |

## Customizing Shortcuts

All shortcuts except `Escape` and `Delete` can be rebound in the **Settings** UI. Nokkvi writes bindings under the `[hotkeys]` section of `config.toml`, and only non-default bindings are written, so you only need to list the ones you want to change.

A direct edit to `config.toml` can also override `escape` or `reset_to_default` ‚Äî the loader accepts those action keys ‚Äî but the in-app editor refuses to rebind them so you keep a way out of every screen.

### Key combo syntax

A binding is a string in the form `"Modifiers + Key"`. Modifiers are optional; the key name is required.

| Part | Values |
| :--- | :--- |
| Modifier | `Shift`, `Ctrl` (or `Control`), `Alt` |
| Key | Any single character (`a`‚Äì`z`, `0`‚Äì`9`, `/`, `-`, `=`, `` ` ``, etc.) |
| Named key | `Space`, `Enter`, `Escape`, `Backspace`, `Tab`, `Up`, `Down`, `Left`, `Right`, `Page Up`, `Page Down`, `Home`, `End`, `Delete`, `Insert`, `F1`‚Äì`F12` |

Multiple modifiers are combined with `+`: `"Ctrl + Shift + S"`.

Key names are case-insensitive. `"shift + l"` and `"Shift + L"` are the same.

### Example

```toml
[hotkeys]
# Rebind slot navigation to J/K (vim style)
slot_list_down = "J"
slot_list_up   = "K"

# Move the visualizer cycle to a function key
cycle_visualization = "F2"

# Use Ctrl+Shift+S to save queue as playlist
save_queue_as_playlist = "Ctrl + Shift + S"
```

<Aside type="tip">
  Enable **Verbose Config** in Settings (or set `verbose_config = "on"` in `config.toml`) to write all default bindings to the file at once ‚Äî useful as a starting point for customization.
</Aside>

<Aside type="caution">
  Some keys may be captured by your window manager before Nokkvi sees them ‚Äî common culprits are `Super`, `Alt + F4`, and certain `Ctrl + Alt` combinations. If a shortcut isn't working, check your compositor or WM keybindings first.
</Aside>

<Aside type="tip">
  To trigger Nokkvi from a WM hotkey (with or without the window focused), use the [`nokkvi` CLI](/guides/cli/) ‚Äî `nokkvi play-pause`, `nokkvi rate +1`, etc. ‚Äî bound through your compositor's exec syntax. When the window isn't focused, enable [`rating_change_notification_enabled`](/reference/config/#rating-reminder) to get a desktop notification confirming each rating change.
</Aside>
